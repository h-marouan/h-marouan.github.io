## 🚀 Projets mis en avant

### 🤖 ROBOT-AI — Assistant industriel multi-agents
Un assistant industriel conçu comme une **entreprise virtuelle pilotée par IA**, reposant sur une architecture **multi-agents** et une approche **hybride Cloud / Local** (fallback).

### 👁️ SMART_VIEWER — Inspection couleur autonome (Raspberry Pi 5 + Pico W)
Système d’inspection industrielle en temps réel : **capture caméra**, **interface tactile PyQt**, **moteurs de correction** et communication **RS485** pour motorisation distante.  
- Architecture Pi5 ↔ RS485 ↔ Pico W
- Optimisations thermiques (LED sync framerate)
- Roadmap IA (détection autonome)

Doc (dans ce repo) : `docs/smart-viewer/README.md`

---

### 🧩 OptiQt — Vision + CNC + génération de G-code
Framework Python modulaire dédié à l’optimisation et au pilotage de systèmes **CNC** et **vision industrielle**.  
- Modules vision (acquisition, traitement, extraction contours)
- Modules CNC (génération G-code, communication série, contrôle mouvement)
- Interface Qt + tests unitaires

Repo : https://github.com/h-marouan/OptiQt  
Doc (dans ce repo) : `docs/optiqt/README.md`
